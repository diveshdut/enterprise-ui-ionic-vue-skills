# Claude Project Instructions

This repository contains shared guidance for enterprise Ionic/Vue UI design
and implementation.

## Working Rules

- Read the repo-local instructions before designing or coding UI.
- Use the design guidance first when a Figma file or other design artifact is
  available.
- Use the workflow-pattern guidance when requirements do not come with a
  design.
- Use the coding standards guidance for any Ionic/Vue implementation or
  refactor.
- Keep the UI mobile-compatible and avoid unnecessary custom styling.
- Prefer reusable, generic enterprise patterns over ad hoc layouts.

## Skill Order

When relevant, load or consult:

1. `enterprise-workflow-design`
2. `ionic-vue-enterprise-patterns`
3. `ionic-vue-coding-standards`
4. `ionic-vue-data-state-patterns`
